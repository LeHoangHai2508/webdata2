CSV File Format

Your CSV file should have the following columns:

  • name (string) - Product name
  • price (integer) - Product price in your currency
  • description (string) - Product description

Example:

name,price,description
Tropical Fruit,15000,Fresh tropical fruits including mango, pineapple, and papaya
Whole Milk,25000,1 liter of fresh whole milk
Pip Fruit,12000,Assorted pip fruits including apples and pears

Note: Make sure the price is a valid integer number.
